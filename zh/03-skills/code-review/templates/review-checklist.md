# 程式碼審查檢查清單

## 安全檢查
- [ ] 沒有硬編碼的憑據或金鑰
- [ ] 所有使用者輸入都做了校驗
- [ ] 使用引數化查詢，防止 SQL 注入
- [ ] 所有會修改狀態的操作都有 CSRF 防護
- [ ] 使用正確的轉義，防止 XSS
- [ ] 受保護的端點有身份驗證檢查
- [ ] 資源訪問有授權檢查
- [ ] 密碼使用安全雜湊演算法（bcrypt、argon2）
- [ ] 日誌中沒有敏感資料
- [ ] 強制使用 HTTPS

## 效能檢查
- [ ] 沒有 N+1 查詢
- [ ] 索引使用合理
- [ ] 在有價值的地方做了快取
- [ ] 主執行緒上沒有阻塞操作
- [ ] 正確使用 async/await
- [ ] 大資料集已經分頁
- [ ] 資料庫連線已做連線池
- [ ] 正規表示式已最佳化
- [ ] 沒有不必要的物件建立
- [ ] 沒有記憶體洩漏

## 質量檢查
- [ ] 函式少於 50 行
- [ ] 變數命名清晰
- [ ] 沒有重複程式碼
- [ ] 錯誤處理合理
- [ ] 註釋解釋的是 WHY，而不是 WHAT
- [ ] 生產環境中沒有 console.log
- [ ] 有型別檢查（TypeScript / JSDoc）
- [ ] 遵循 SOLID 原則
- [ ] 正確應用設計模式
- [ ] 程式碼具備自解釋性

## 測試檢查
- [ ] 已編寫單元測試
- [ ] 覆蓋了邊界情況
- [ ] 測試了錯誤場景
- [ ] 有整合測試
- [ ] 覆蓋率大於 80%
- [ ] 沒有不穩定測試
- [ ] 外部依賴已做 mock
- [ ] 測試名稱清晰
