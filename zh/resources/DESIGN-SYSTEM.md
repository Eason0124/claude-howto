# Claude How To - 設計系統

## 視覺識別

### 圖示設計概念：指南針 + 程式碼括號

Claude How To 的圖示使用**指南針加 `>` 程式碼括號**來表示“在程式碼中導航”：

```
     N (green)
     ▲
     │
W ───>─── E     Compass = Guidance/Direction
     │          > Bracket = Code/Terminal/CLI
     ▼
     S (black)
```

它帶來了以下效果：
- **視覺清晰**：一眼就能看出這是“程式碼導航指南”
- **象徵意義**：指南針 = 找到方向；`>` = 程式碼/終端
- **可擴充套件性**：從 16px 到 512px 都能正常工作
- **品牌一致性**：以極簡配色匹配開發者工具氣質

## 色彩系統

### 配色

| 顏色 | 十六進位制 | RGB | 用途 |
|-------|-----|-----|-------|
| 黑色（主色） | `#000000` | 0, 0, 0 | 主描邊、文字、南針 |
| 白色（背景） | `#FFFFFF` | 255, 255, 255 | 淺色背景 |
| 灰色（輔助） | `#6B7280` | 107, 114, 128 | 次要刻度、次級文字 |
| 亮綠色（強調） | `#22C55E` | 34, 197, 94 | 北針、中心點、強調線 |
| 接近黑色（深色背景） | `#0A0A0A` | 10, 10, 10 | 深色模式背景 |

### 對比度（WCAG）

- 黑底白字：**21:1** AAA
- 灰底白字：**4.6:1** AA
- 綠底白字：**3.2:1**（僅用於裝飾，不用於正文）
- 深色背景白字：**19.5:1** AAA

### 強調色規則

**亮綠色（#22C55E）僅用於高亮：**
- 指南針北針
- 中心點
- 強調下劃線/邊框
- 絕不作為背景色
- 絕不用於正文

## 圖示細節

### 指南針規格

指南針圖形由以下幾何元素組成：

```
Element             | Stroke/Fill    | Color
--------------------|----------------|------------------
Outer ring          | 3px stroke     | Black / White (dark mode)
North tick          | 2.5px stroke   | Black / White (dark mode)
Other cardinal ticks| 2px stroke     | Gray / White 50% (dark mode)
Intercardinal ticks | 1.5px stroke   | Gray / White 40% (dark mode)
North needle        | filled polygon | #22C55E (always green)
South needle        | filled polygon | Black / White (dark mode)
> bracket           | 3px stroke     | Black / White (dark mode)
Center dot          | filled circle  | #22C55E (always green)
```

### 尺寸遞進

```
16px  → 僅環 + 指標 + 箭頭（最簡版）
32px  → 增加方位刻度
64px  → 增加中間方位刻度
128px → 全細節，所有元素都清晰
256px → 最高細節，較粗描邊
```

- 所有文字都滿足 WCAG AA（最低 4.5:1）
- 綠色強調色只用於裝飾，不承擔資訊傳達

## 自定義

### 更換強調色

如果你想建立不同強調色的變體：

1. 把所有 `#22C55E` 替換成你的強調色
2. 確保裝飾性元素的對比度仍高於 3:1
3. 保持黑/白/灰結構不變

### 縮放

```css
svg {
  width: 256px;
  height: 256px;
}
```

SVG 會透過 `viewBox` 自動縮放，不需要額外變換。
