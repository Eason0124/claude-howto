# EPUB 構建指令碼

## 功能

這個指令碼會把整份指南構建成 EPUB 電子書，方便離線閱讀。

## 需求

- Python 環境
- `uv`
- 相關依賴

## 快速開始

最簡單的方式是直接讓 `uv` 幫你處理依賴：

```bash
uv run scripts/build_epub.py
```
